Description

porfs (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SPORFS, DPORFS, CPORFS, and ZPORFS. porfs improves the computed solution to a system of linear equations when the coefficient matrix is Hermitian positive definite, and provides error bounds and backward error estimates for the solution.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixA. The type of values is obtained through the value_type meta-function typename value_type<MatrixA>::type.
