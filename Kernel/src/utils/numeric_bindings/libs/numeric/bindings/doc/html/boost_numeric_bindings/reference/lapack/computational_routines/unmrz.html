unmrz
Prototype

Description

unmrz (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SORMRZ, DORMRZ, CUNMRZ, and ZUNMRZ. unmrz overwrites the general complex M-by-N matrix C with

SIDE = 'L' SIDE = 'R' TRANS = 'N': Q * C C * Q TRANS = 'C': Q*H * C C * Q*H

where Q is a complex unitary matrix defined as the product of k elementary reflectors

Q = H(1) H(2) . . . H(k)

as returned by ZTZRZF. Q is of order M if SIDE = 'L' and of order N if SIDE = 'R'.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixA. The type of values is obtained through the value_type meta-function typename value_type<MatrixA>::type.
